# АвтоЗапчасти — Система управления складом

## Стек
- Django 4.x / 5.x
- Bootstrap 5 + Bootstrap Icons
- SQLite (легко переключить на PostgreSQL)
- openpyxl (экспорт чеков в Excel)
- Chart.js (графики на дашборде)

## Установка и запуск

```bash
# 1. Установить зависимости
pip install django openpyxl pillow

# 2. Применить миграции
python manage.py makemigrations account catalog receipt dashboard panel
python manage.py migrate

# 3. Создать суперпользователя
python manage.py create_superuser
# Логин: admin | Пароль: admin123

# 4. Заполнить категории (BMW, Mercedes, Audi, VW, Toyota и др.)
python manage.py seed_categories

# 5. Заполнить демо-данными (пользователи, товары, чеки, поставки)
python manage.py seed_demo

# 5. Создать папку для аватарки и положить туда avatar.png
mkdir -p static/src
# Скопировать любое изображение как static/src/avatar.png

# 6. Запустить сервер
python manage.py runserver
```

Открыть в браузере: http://127.0.0.1:8000

## Структура приложений

| Приложение | Назначение |
|------------|-----------|
| `account`  | Авторизация, роли (продавец / сортировщик / администратор), настройки |

| `dashboard`| Аналитика, графики, уведомления о низком остатке |
| `panel`    | Самописная админ-панель (CRUD для всего) |

## Функционал

### Каталог
- Поиск по названию и описанию
- Фильтрация по категории/подкатегории (чеклист с галочками)
- Сортировка: А–Я, Дешевле, Дороже
- Корзина с добавлением и изменением количества

### Чеки
- Формирование из корзины
- Кнопка "Провести" — списывает со склада, увеличивает счётчик продаж
- Экспорт в Excel с красивым оформлением
- Список чеков с сортировкой по дате и сумме

### Аналитика
- График продаж по дням (последние 30 дней)
- Помесячная выручка (bar chart)
- Выручка за месяц и за год
- Топ-5 продаваемых и непродаваемых товаров
- Блок уведомлений (остаток < 3 шт.)

### Панель администратора (/admin-panel/)
- CRUD: пользователи, категории, подкатегории, товары, чеки, поставки
- Установка скидки на товар (в %)
- Создание поставок с позициями
